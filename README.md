
# ***hairshop***
<br/>
지정된 미용실과 소비자를 연결하여 예약과 관리를 할 수 있는 프로그램입니다.
<br/>
GUI를 통해 클릭으로 예약과 관리를 진행하며, 
<br/>
비트맵 이미지를 이용하여 이미지를 처리를 하였으며,
<br/>
파일을 사용하여 데이터를 처리하였습니다.
<br/>
중점적으로 프로그램이 구동하는 정확성과 사용자가 예약하는데에 있어서 이미지처리와 파일처리를 집중적으로 할 수 있도록 하였습니다.
<br/>
